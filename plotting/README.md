# Plotting

Create accuracy plots from YAML config.

## Usage

```bash
python3 plotting/create_main_plot.py plotting/func_corr/main/config.yaml
python3 plotting/create_main_plot.py plotting/diamond/main/config.yaml
```

## Setup

Edit `plotting/func_corr/main/config.yaml` or `plotting/diamond/main/config.yaml` with paths to your results files.
