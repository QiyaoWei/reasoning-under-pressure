# Contributing to Reasoning Under Pressure

Thank you for considering a contribution! We welcome contributions of any kind - bug fixes, enhancements, documentation improvements, or feedback.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/reasoning-under-pressure.git`
3. Install dependencies: `pip install -e . && pip install -r requirements.txt`
4. Create a branch for your changes: `git checkout -b feature/your-feature-name`

## Code Style

We use pre-commit hooks to maintain code quality:

```bash
pip install pre-commit
pre-commit install
pre-commit run --all-files
```

## Pull Requests

When submitting a PR:
- Write clear commit messages
- Update documentation if needed
- Add tests for new functionality where applicable
- Ensure all existing tests pass

## Reporting Issues

Please include:
- A clear description of the issue
- Steps to reproduce
- Expected vs actual behavior
- Environment details (Python version, OS, etc.)
